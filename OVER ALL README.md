# CodeAlpha Cyber Security Internship Projects

## 📌 Project Overview

This repository contains all the cybersecurity internship projects completed as part of the CodeAlpha Cyber Security Internship program.

The projects focus on:
- Network Security
- Packet Analysis
- Intrusion Detection
- Secure Coding Practices
- Phishing Awareness
- Cybersecurity Monitoring

The repository includes practical cybersecurity projects demonstrating real-world attack detection, vulnerability analysis, and security awareness concepts.

---

# 🚀 Tasks Included

## ✅ Task 1 — Basic Network Sniffer

A Python-based packet sniffer that captures and analyzes live network traffic.

### Features
- Live packet capturing
- Source and destination IP analysis
- TCP/UDP/ICMP detection
- Packet payload inspection
- Real-time traffic monitoring

### File
```bash
sniffer.py
```

---

## ✅ Task 2 — Phishing Awareness Training

A cybersecurity awareness presentation explaining phishing attacks, fake websites, social engineering, and online safety best practices.

### Topics Covered
- Types of phishing attacks
- Fake email detection
- Fake website identification
- Social engineering tactics
- Real-world phishing examples
- Interactive awareness quiz

### Files
```bash
Phishing_Awareness_Training.pptx
Phishing_Awareness_Training.pdf
```

---

## ✅ Task 3 — Secure Coding Review

A Python-based secure coding project demonstrating SQL Injection vulnerabilities and secure coding remediation techniques.

### Features
- Vulnerable login system
- SQL Injection demonstration
- Secure login implementation
- Parameterized queries
- Secure coding analysis

### Files
```bash
vulnerable_login.py
secure_login.py
```

---

## ✅ Task 4 — Intrusion Detection System (IDS)

Python-based IDS projects for monitoring network traffic and detecting suspicious activity.

### Included IDS Projects
- Basic IDS
- GUI IDS
- Packet Logger IDS

### Features
- ICMP Ping detection
- TCP SYN scan detection
- Real-time alerts
- Packet logging
- GUI dashboard monitoring

### Files
```bash
ids.py
gui_ids.py
logger_ids.py
ids_alerts.txt
```
